## Running the Study

```bash
# List study structure
ultra study list ai-code-review-agent-study

# Run a single analysis
ultra study run ai-code-review-agent-study 01 pr-agent

# Run all analyses (requires cloning repos)
ultra study run-all ai-code-review-agent-study

# Run in resumable loop mode
ultra study run-loop ai-code-review-agent-study --batch-size 3

# Check status
ultra study status ai-code-review-agent-study
```

## Note on GitHub Repos

For the 10 GitHub repo sources, you need to clone them locally first. The `ultra study run` command will need to be pointed to local clones. See `study-init.yml` for the expected source URLs.

You can prepare the repo sources by cloning them:

```bash
cd studies/ai-code-review-agent-study/sources
git clone https://github.com/The-PR-Agent/pr-agent
git clone https://github.com/reviewdog/reviewdog
# ... etc for each repo
```
